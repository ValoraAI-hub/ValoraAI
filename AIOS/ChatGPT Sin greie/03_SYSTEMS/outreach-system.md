Goal:
Increase reply rate and booked meetings.

Target:
Recruiters reaching out to candidates or clients via LinkedIn/email.

System:

Input:
- Name
- Role
- Company
- Context (if available)

Process:
1. Identify angle (relevance, curiosity, opportunity)
2. Generate short personalized message
3. Generate 2 follow-ups
4. Keep tone human and direct

Output:
- Message ready to send (no edits needed)

Rules:
- Max 2–3 lines per message
- No fluff
- No corporate language
- Must feel like 1-to-1 message

Success metrics:
- Reply rate (%)
- Positive replies
- Booked calls

Benchmarks:
- Bad: <5%
- OK: 5–10%
- Good: 10–20%
- Strong: 20%+

Current hypothesis:
Short, slightly curious, informal messages outperform long structured ones.

Test plan:
- Test 3 variations per batch
- Send minimum 10–20 messages per variation
- Track results in insights.md
